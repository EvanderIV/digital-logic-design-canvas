Lab Information

Objective:

The purpose of this laboratory is to familiarize the student with combinational logic design and implementation in hardware. The student will design a Full Adder using a truth table to correctly specify the needed functionality and then simplify it using a Karnaugh map (K-map). This simplified expression will then be implemented using TTL parts available in the laboratory on the CADET II.

Key Words/Abbreviations:

LSB, MSB, schematic, algorithm, bit, digit, Boolean expression, Boolean algebra, SOP, CIN, COUT, K-map
