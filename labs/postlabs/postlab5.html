Lab Information

Objective:

The purpose of this laboratory is to familiarize the student with Multiplexers and Decoders. These MSI (Medium Scale Integration) components will be used to implement one combinational logic network and several multiplexed communication channels.

Key Words/Abbreviations:

MUX, DECODER, low-active output, D flip Flop (DFF), select lines, latched
