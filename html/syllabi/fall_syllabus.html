Course Description

This course provides a comprehensive introduction to the principles and practices of digital logic design. Topics include number systems, Boolean algebra, logic gates, analysis and design of combinational and sequential logic circuits, Karnaugh maps, and the use of modern design tools like Quartus II. Students will gain hands-on experience through laboratory assignments.

Learning Outcomes

Upon successful completion of this course, students will be able to:

  • Interpret and perform math operations on both positive & negative binary numbers
  • Design two level combinational logic circuits using Karnaugh maps
  • Design logic circuits with higher level building blocks such as Multiplexers, Decoders, Registers, Counters and Memory devices including finite state machines
  • Effectively use available software for implementation, simulation & verification of logic designs
  • Demonstrate proficiency in the laboratory connecting, debugging, and verifying logic circuits
