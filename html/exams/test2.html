Test #2

EGCP 1010: Digital Logic Design

Test Information

This test assesses your knowledge of complex digital logic, as well as your ability to create and analyze finite state machines. This test will cover chapters 8.3, 11-14 of the textbook.

Study Guide

Topic / Skill Example Task / Reference
Feedback Systems – Stable vs. Unstable Given a feedback circuit, determine the output (SG 1 p. 318).
SR latch behavior (hold, reset, set, not-allowed) Given an input waveform, find the output waveform (SG 5 p. 319).
Gate Delay effects Given an input waveform, find the output waveform considering gate delay (Fig 8-5 p. 223).
Basic Flip-Flop Functions (D & JK with preset/clear) Given an input waveform, find the output waveform (SG 8 p. 320). Know definitions: Tsu, Thold, Metastability.
Shift Registers Know the CedarLogic "SHIFT" component (Fig 12-7 p. 358, Fig 12-10 p. 360).
Counter Design Design a counter that counts in the sequence 0,2,4,3,1,5,7,0...
Finite State Machine (FSM) Analysis From a circuit schematic, complete a full analysis to a state flow graph (See Problem 13.8).
FSM Concepts (Mealy vs. Moore) How many flip-flops are needed for 12 states? With 2 inputs, how many arcs leave each state?
FSM Design from Word Problem Design a sequence detector for 0111, or a sequential Adder/Subtractor (like 14.37, 14.38).
Lab Related Material Know all definitions on memory handout (e.g., Multiplexed communication, Serial-to-Parallel, comparator, etc.).
