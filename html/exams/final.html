Final Exam

EGCP 1010: Digital Logic Design

Test Information

This test broadly assesses your knowledge of digital logic covered throughout this course, as well as your ability to perform advanced binary arithmetic (including negative numbers) and analyze a small computer. This test is comprehensive, and will cover chapters 1-14 & 18 of the textbook.

Study Guide

Topic / Skill Example Task / Reference
CedarLogic Counting Registers Given a configured circuit, describe its behavior or create a circuit for a given count sequence (e.g., 3,4,5...B,3).
Small Computer Function Provide commands to implement functions like A'C->B or A+B'->C.
Negative Number Representations Negate numbers in S&M, 1's, and 2's complement. Identify the value of a bit pattern (e.g., 10110) in each system.
Two's Complement Arithmetic Perform addition of several numbers and indicate if overflow occurred.
Full FSM Design Process Go from word description to state graph, NST, K-maps, and final circuit.

Sequential Circuits Understand function of SR, D, JK flip-flops. Analyze and design sequential circuits. Know Mealy vs. Moore.
Lab & Hardware Concepts What is Fanout, max FSM clock speed, TTL, bit slicing? ROM sizing (address vs data lines).
