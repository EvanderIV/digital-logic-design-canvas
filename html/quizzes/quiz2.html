Quiz #2: Conversion thru Algebra

EGCP 1010: Digital Logic Design

Quiz Information

This quiz tests your ability to convert between numbers of different bases, and perform binary arithmetic. Familiarize yourself with the Laws and Theorems of Boolean Algebra.

Study Guide

Concept of Positional Numbers

  • Convert from base 10 to base 8, then use bit groups to convert to base 2 and base 16.
  • Apply the definition of a positional number to convert base 2, 8, or 16 to base 10.
  • Be able to interpret numbers in other bases. Example: 345 = 3*51 + 4*50 = 1910

Arithmetic in Other Bases

  • Be able to Add, Subtract, Multiply and Divide Binary numbers.
  • ASCII – Stands for American Standard Code for Information Interchange and all information in a computer or other digital device is stored only in 1's and 0's.

Boolean Algebra to Gate Level Schematic

  • Given a Boolean algebra expression, draw the Gate level schematic. Example: F=[A'B + (CD'+F)']
  • Given a Gate level schematic, find the Boolean algebra expression.

Simplification of Boolean Logic

  • Be able to simplify Boolean algebra expressions.
  • Use Demorgan's theorem to find the Complement of a Boolean expression.
